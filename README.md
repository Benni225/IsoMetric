# IsoMetric
IsoMetric is a tilegame-engine for HTML5 written in TypeScript. It is still in pre-alpha.
Some of the main features are already implemented. 

## Example

Just try the examples in the examplefolder.

## Features
* Layering
* Scrolling
* Maps
* Mapediting
* Tilesets
* Inputs
* Sprites
* Colliding
